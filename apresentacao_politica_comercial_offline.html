Offline + CSV + Politica Comercial

Apresentacao comercial em HTML para distribuir sem internet.

Este arquivo funciona sozinho, abre direto do disco e usa uma planilha CSV como base local de dados para a politica de precos. A regra principal do backend foi espelhada no navegador para manter o atendimento comercial coerente mesmo sem servidor.

Regra Comercial

preco_base_politica = max(preco_planilha, valor_obtido) + sobrepreco preco_final = max(preco_base_politica, preco_digitado_corretor) bloqueio: se preco_digitado_corretor > 0 e preco_digitado_corretor < preco_base_politica entao retorna "Valor abaixo da politica comercial"

Planilha Como Banco

Colunas esperadas: empreendimento, unidade, garantido_minimo, preco, sobrepreco, is_maximo, prosoluto_minimo.

A base importada fica guardada no navegador via localStorage. Se sua planilha estiver no Excel, salve como CSV UTF-8 antes de distribuir.

Base de Dados Local

Importe a planilha, filtre as unidades e escolha a linha que servira de base para a proposta.
